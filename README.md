# Nuxt Todo App

タスク管理アプリケーション。

## Setup

1. リポジトリのクローン
```bash
git clone https://github.com/takashi823/nuxt-todo-app.git

cd nuxt-todo-app
```

2. 依存ライブラリのインストール

```bash
npm install
```

## Development Server

3. 開発サーバーの起動

[http://localhost:3000](http://localhost:3000) でアクセス可能

```bash
npm run dev
```
